The PBR database v0.1

This is an API where you can look up physically based values for your CG materials.

It is meant to be consumed by DCC applications in the form of plug-ins, but there is also a website in development where you can easily browse and look up values at: https://physicallybased.info

Contact me for support, feedback, and suggestions: https://twitter.com/AntonPalmqvist
