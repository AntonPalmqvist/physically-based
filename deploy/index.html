The PBR Database API v0.1

This is an API where you can look up physically based values for your CG materials.

It is meant to be consumed by DCC applications in the form of plug-ins, but there is also a website where you can easily browse and look up values at: https://physicallybased.info

Feel free to contact me for support, feedback, and suggestions: https://twitter.com/AntonPalmqvist

Endpoints

https://api.physicallybased.info/materials

https://api.physicallybased.info/lightsources
