The PBR Database API

This is an API where you can look up physically based values for your CG materials.

It is meant to be consumed by DCC applications in the form of plug-ins, or any other form that makes sense.

For additions or changes, please make a Pull Request on Github https://github.com/AntonPalmqvist/physically-based-api or send me a message through this form.
